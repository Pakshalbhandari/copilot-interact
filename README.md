# Copilot Interact

Invoke GitHub Copilot with custom prompt templates and sequential chains.

## Features

- Execute single prompts with custom templates
- Chain multiple prompts sequentially
- Create predefined chains or custom chains on the fly

## Usage

1. Create a `prompts` folder in your workspace
2. Add `.md` files with your prompt templates
3. Optionally create `.chain.json` files for predefined chains
4. Select code and run `Copilot Interact` commands

## Commands

- `Copilot Interact: Invoke with Custom Prompt` - Run a single prompt
- `Copilot Interact: Execute Sequential Prompts` - Run a chain of prompts
